# Data Dictionary – Interrogation Dataset

The dataset columns are stored using generic field names (`field1`, `field2`, …).
This document explains the semantic meaning of each attribute based on the
original Italian dataset description.

|Field Name | Italian Column         | English Meaning                           |
|-----------|------------------------|-------------------------------------------|
| field1    | id                     | Unique identifier of the record           |
| field2    | data                   | Date of arrival at the hotel              |
| field3    | nome                   | First name                                |
| field4    | cognome                | Last name                                 |
| field5    | targa_auto             | Car license plate                         |
| field6    | citta                  | City of origin                            |
| field7    | partenza               | Time the person left the hotel            |
| field8    | ritorno                | Time the person returned to the hotel     |
| field9    | capelli                | Hair color                                |
| field10   | numero_scarpe          | Shoe size                                 |
## Notes
- The dataset was provided with generic column names (`field1`, `field2`, …).
- Column semantics were inferred from the exercise description and data values.
- Column names were intentionally not renamed to preserve the original dataset structure.
